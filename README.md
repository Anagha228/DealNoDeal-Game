# Deal or No Deal (VBA Edition)

A recreation of the classic *Deal or No Deal* game built using **Visual Basic for Applications (VBA)**.

This project simulates the television game show where players choose briefcases containing hidden cash values and decide whether to accept offers from the banker or continue playing for the top prize.

---

## Features

- Randomized briefcase values
- Interactive game flow
- Banker offer system
- Case elimination tracking
- Win/Lose outcomes
- Built entirely in VBA

---

## Technologies Used

- Visual Basic for Applications (VBA)
- Microsoft Excel

---

## How to Run

1. Open the Excel workbook containing the VBA project.
2. Enable macros when prompted.
3. Press the play button

---

## Game Rules
1. Select your personal briefcase.
2. Open a set number of remaining cases each round.
3. After each round, the banker will make an offer.
4. Choose:
    - Deal → accept the banker’s offer
    - No Deal → continue the game
5. Reach the end to discover your case value.
